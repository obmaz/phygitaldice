# Dice Arena Companion

Flutter 기반의 하이브리드 주사위 카드게임 컴패니언 앱 프로토타입입니다.

## 지원 플랫폼

- Android
- iOS

## 현재 구현 범위

- 캐릭터 선택
- 전용 주사위 면 표시
- 추가 스킬 2개 선택
- 캐릭터 현재 체력 조작
- 상태 마커 토글
- 게임 준비 체크리스트

실제 주사위 굴림, 카드 사용, 피해 계산은 오프라인 플레이에서 처리합니다.

## 실행

```sh
flutter pub get
flutter run
```

## 검증

```sh
flutter analyze
flutter test
```
